# Tellurium
Tellurium is a universal redirect/SSL bypass for native UE HTTP games. (designed for Fortnite) <br>

# Configuring Tellurium

## Changing Tellurium's Backend URL.
To change Tellurium's backend URL to yours, go to ``Tellurium/Options.h`` and change the ``Backend`` variable to your backend's URL.

## Build for a Hybrid server
Go to ``Tellurium/Options.h``, and change ``UrlSet`` to Hybrid.

## Build for a Dev server
Go to ``Tellurium/Options.h``, and change ``UrlSet`` to Dev.

## Redirecting everything
Go to ``Tellurium/Options.h``, and change ``UrlSet`` to All.

## Enabling Console for debugging
Go to ``Tellurium/Options.h``, and change ``Console`` to true.
